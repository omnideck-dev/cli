STEP 2 OF 2
Install Docker
Nothing starts until you press Enter.

What happens next
    1. Open Docker's official download page.
    2. Download the installer for your Mac and follow the instructions on screen.
    3. Start Docker Desktop, return to Omnideck, and check again.

Permission or password
The Docker installer may ask for your Mac password so macOS can add the app.
Omnideck does not see or store it.

Before you continue
Docker Desktop can require a paid subscription at larger companies. If this is a work computer,
ask your company whether it already provides Docker Desktop.
